This is a Dropwizard project (https://www.dropwizard.io)

Requirements to run the project:

1- Install Java8 https://www.java.com/es/download/faq/java8.xml
2- Install Maven https://maven.apache.org/

To run this project:

1- Go to project directory
2- mvn package
3- java -jar target/qa-test-1.0-SNAPSHOT.jar server
